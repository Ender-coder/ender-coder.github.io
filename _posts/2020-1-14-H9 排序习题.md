### 9、将哨兵放在R[n]中，被排序的记录存放在R[1...n-1]中，重写直接插入排序算法。
```cpp
void InsertSort(DataType R[], int n){
    for(int i = n-2; i > 0; i--){
        R[n] = R[i];
        for(int j = i+1; R[j] > R[n]; j++){
            R[j-1] = R[j];
        }
        R[j-1] = R[n];
    }
}
```
